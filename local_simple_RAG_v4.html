User Guide

What is Local RAG?

RAG (Retrieval-Augmented Generation) restricts the AI to answer using only specific information you provide, rather than its general training data.

How to use Knowledge Base

  1. Add Content: Paste text directly into the "Knowledge Base" box in the sidebar, or use the Upload File button to load files (.txt, .md, .csv, .docx, .pdf).
  2. Verify Status: The indicator below the text box will turn Green and say "RAG Active".
  3. Ask Questions: Type your question. The Generator will now ignore its internal knowledge and use only your uploaded text to answer.

Reviewer Mode

Enable the Reviewer in the sidebar to double-check the Generator.

  • The Reviewer acts as an auditor.
  • It reads your Context, the Generator's Answer, and checks for hallucinations (things made up that weren't in the text).

Optimization & Limits

For the best response time (under 10s), keep these limits in mind:

  • Text Volume: Ideal is < 10 pages (~20k chars). Max is ~15 pages. Most local models have an 8k token limit; exceeding this will cause truncation or errors.
  • File Size: Keep text files under 100KB and PDFs/DOCX under 5MB to ensure the browser doesn't freeze during parsing.

Connection Setup

Ensure your local server is running and CORS is enabled.

  • Ollama OLLA_ORIGINS="*" ollama serve
  • LM Studio Enable CORS in Server tab settings.
